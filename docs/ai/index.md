# AI & Codegen Overview

Ellithium 3.0.0-Beta introduces a powerful suite of AI features designed to make your automation code resilient, easier to generate, and extremely intelligent.

## Core Capabilities

- **AI Self-Healing:** A cascading 3-Tier strategy system that automatically repairs broken element locators at runtime without breaking your test execution.
- **AI Codegen Recorder:** A robust in-browser interaction recorder that generates fully functional TestNG classes or reusable Page Objects from your manual actions. No more writing boilerplates.
- **AI Engine (Natural Language to Code):** A live context generator that can execute and generate test automation scripts dynamically straight from natural language descriptions (JSON or plain text format) or while you're directly in a test.
- **AI Test Data Generator:** Generates diverse edge-case payloads (Fuzzing) to comprehensively test your system boundaries based simply on a schema description.
